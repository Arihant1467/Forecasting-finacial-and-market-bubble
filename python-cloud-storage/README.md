### How to Run Python scripts
We will use *Python-3* for our purpose



#### To Get started
```
make activate
make install-tools
```


#### Activate virtual env
```
make activate
```

#### Deactivate virtual env
```
make deactivate
```

#### To install packages
```
make install-tools
```

#### To install any tools inside Virtual Env
```
./bin/pip install package-name

```
